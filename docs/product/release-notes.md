## V14

- 发布时间：2026-03-24
- 发布提交：`51ed313`
- 发布分支：`main`

### 本次版本重点

- 完成账号体系第一阶段接入，支持注册、登录、退出和基础个人资料读取。
- 完成 Supabase 云端计划保存能力，支持把旅行计划保存到云端并重新读取。
- 网站结构升级为 `首页 / 行程库 / 功能页` 三页架构。
- 完成品牌接入，统一使用 `行旅派 | GoPace` 命名、logo 和品牌色。
- 首页与行程库完成一轮产品化 UI 升级。

### 主要更新说明

#### 1. 账号与云端能力

- 接入 Supabase Auth，支持邮箱注册和登录。
- 接入 Supabase Postgres，支持旅行计划云端保存与读取。
- 支持计划归档、取消归档、复制为新计划、删除。
- 支持当前规划与云端计划绑定，便于继续编辑。

#### 2. 页面结构升级

- 首页改为账号与概览页。
- 新增独立的“行程库”页面，用于集中管理全部旅行计划。
- 功能页保留为核心规划工作区。
- 顶部导航统一为 `首页 / 行程库 / 功能页`。

#### 3. 首页优化

- 登录区与注册区整合为切换式表单。
- 登录后首页改为概览视图，不再重复占用大块账号面板。
- 新增当前计划焦点区，可显示当前绑定计划的摘要。
- 新增最近计划列表，方便快速继续编辑。
- 品牌 hover 卡支持查看当前账号信息和计划数量。

#### 4. 行程库升级

- 行程库从首页中独立拆分为正式页面。
- 计划卡支持显示日期、人数字段、更新时间。
- 新增摘要信息展示，包括：
  - 行程天数
  - 地点数量
  - 节点数量
  - 当前状态
- 卡片支持继续编辑、复制、归档、取消归档、删除。

#### 5. 品牌视觉升级

- 接入 `行旅派 | GoPace` 品牌体系。
- 顶部导航改为品牌 logo + 名称组合。
- favicon 切换为品牌符号。
- 页面主色统一到品牌色板：
  - `#3b5944`
  - `#cd8147`
  - `#d6ad78`
  - `#f2eadb`
  - `#2b353f`

### 当前可用能力

- 本地规划多日旅行行程
- 账号注册与登录
- 云端保存旅行计划
- 查看、载入、复制、归档、删除计划
- 首页查看当前计划与最近计划
- 行程库集中管理全部计划

### 下一步建议

- 完善账户页
- 增强行程库详情抽屉或详情页
- 增加分享、导出和协作功能
- 完善多人共同编辑同一计划的数据结构
